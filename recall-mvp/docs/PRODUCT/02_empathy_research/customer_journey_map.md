# Evermore Customer Journey Map

---

## Journey Overview

Two parallel journeys that intersect:

1. **Senior Journey**: From first introduction to regular use
2. **Family Journey**: From discovery to ongoing engagement

---

## Senior Journey

### Stage 1: Introduction

| Dimension | Description |
|-----------|-------------|
| **Trigger** | Family member gifts Evermore, senior center demo, care manager recommendation |
| **Touchpoints** | Physical device setup, family assistance, first voice interaction |
| **Actions** | Listens to introduction, says first words to Evermore |
| **Thoughts** | "Will this be complicated?" "Can I really just talk?" |
| **Emotions** | Skeptical, curious, slightly anxious |
| **Pain points** | Fear of technology, embarrassment about needing help |
| **Opportunities** | Immediate warm greeting, no setup required, voice-only |

### Stage 2: First Conversation

| Dimension | Description |
|-----------|-------------|
| **Trigger** | Device is ready, senior is alone |
| **Touchpoints** | Voice prompt, Evermore initiates |
| **Actions** | Responds to gentle opening question, tells first story |
| **Thoughts** | "This is actually easy." "It's listening to me." |
| **Emotions** | Surprised, pleased, engaged |
| **Pain points** | Uncertainty about what to say, fear of judgment |
| **Opportunities** | Scaffolded questions, immediate positive feedback, patience |

### Stage 3: Building Trust

| Dimension | Description |
|-----------|-------------|
| **Trigger** | Evermore references previous conversation |
| **Touchpoints** | Daily prompts, memory callbacks |
| **Actions** | Returns for more sessions, shares deeper stories |
| **Thoughts** | "It remembers what I said." "Maybe I can share more." |
| **Emotions** | Warming, trusting, valued |
| **Pain points** | Occasional technical issues, forgetting to engage |
| **Opportunities** | Memory continuity, gentle proactive prompts |

### Stage 4: Regular Use

| Dimension | Description |
|-----------|-------------|
| **Trigger** | Daily routine, photo triggers, family prompts |
| **Touchpoints** | Voice sessions, photo sharing, family dashboard notifications |
| **Actions** | 3-5 sessions per week, shares photos, references past stories |
| **Thoughts** | "This is part of my day now." "My family can see this." |
| **Emotions** | Connected, purposeful, heard |
| **Pain points** | Repetition, occasional confusion about features |
| **Opportunities** | Story arcs, chapter creation, family engagement loops |

### Stage 5: Legacy Awareness

| Dimension | Description |
|-----------|-------------|
| **Trigger** | Family shares appreciation, sees story collection growing |
| **Touchpoints** | Story export, printed book, family acknowledgment |
| **Actions** | Reviews captured stories, adds context, feels accomplishment |
| **Thoughts** | "My life is being recorded." "This will live on." |
| **Emotions** | Pride, peace, satisfaction |
| **Pain points** | Finite topic space, energy/health changes |
| **Opportunities** | Chapter organization, guided prompts for new topics |

---

## Family Journey

### Stage 1: Discovery

| Dimension | Description |
|-----------|-------------|
| **Trigger** | Article, friend recommendation, parent's health scare, holiday gift search |
| **Touchpoints** | Website, reviews, social media |
| **Actions** | Researches, compares options, evaluates for parent |
| **Thoughts** | "Is this something my parent would actually use?" |
| **Emotions** | Hopeful, skeptical, worried about adoption |
| **Pain points** | Uncertainty about parent's willingness |
| **Opportunities** | Clear value prop, testimonials from seniors, easy trial |

### Stage 2: Purchase and Setup

| Dimension | Description |
|-----------|-------------|
| **Trigger** | Decision to try |
| **Touchpoints** | Checkout, gift messaging, setup instructions |
| **Actions** | Purchases subscription, schedules setup call with parent |
| **Thoughts** | "I hope this isn't too complicated for them." |
| **Emotions** | Optimistic, anxious about parent adoption |
| **Pain points** | Remote setup challenges, parent's tech anxiety |
| **Opportunities** | Concierge onboarding, 1-click setup, phone support |

### Stage 3: Monitoring Engagement

| Dimension | Description |
|-----------|-------------|
| **Trigger** | First session notification |
| **Touchpoints** | Dashboard, email digests, mobile notifications |
| **Actions** | Checks dashboard, reads story summaries |
| **Thoughts** | "It's working!" "I didn't know that about mom." |
| **Emotions** | Relieved, surprised, connected |
| **Pain points** | Notification fatigue, privacy concerns |
| **Opportunities** | Configurable alerts, weekly digests, story highlights |

### Stage 4: Active Participation

| Dimension | Description |
|-----------|-------------|
| **Trigger** | Wants to deepen engagement |
| **Touchpoints** | Photo uploads, prompt suggestions, comments |
| **Actions** | Sends old photos to trigger conversations, suggests topics |
| **Thoughts** | "I can actually contribute to this." |
| **Emotions** | Engaged, collaborative, connected to parent |
| **Pain points** | Time constraints, feature complexity |
| **Opportunities** | Simple photo upload, pre-written prompts, easy sharing |

### Stage 5: Legacy Retrieval

| Dimension | Description |
|-----------|-------------|
| **Trigger** | Holiday, parent milestone, parent's decline/passing |
| **Touchpoints** | Story export, printed book, sharing with extended family |
| **Actions** | Downloads archive, orders printed book, shares with siblings |
| **Thoughts** | "This is worth so much more than I expected." |
| **Emotions** | Gratitude, sadness, connection |
| **Pain points** | Export complexity, emotional difficulty |
| **Opportunities** | One-click book creation, archive formats, family sharing |

## Journey Intersection Points

| Moment | Senior Action | Family Action | Opportunity |
|--------|---------------|---------------|-------------|
| Photo trigger | Sees photo, tells story | Sends photo | Real-time connection |
| Story capture | Tells a meaningful story | Receives notification | Immediate feedback |
| Safety event | Evermore intervenes | Alert received | Peace of mind |
| Milestone | 50th story captured | Celebration notification | Shared accomplishment |
